# Getting Started

Welcome to the **Genki** documentation!

If you are new to Genki, we recommend that you read our `documentation`. This documentation will guide you through everything you need to know about Genki and how to use Ghost.

If you have questions about anything related to Genki, feel free to open a ticket on [GitHub issues](https://github.com/genkimedia/genki-docs/issues/new).

## Genki Introduction

**[Genki](https://blog.genki.site/)** features [EOS](https://eosnetwork.com?ref=genki-docs), [Telos](https://telos.net?ref=genki-docs), [Wax](https://.wax.io?ref=genki-docs) and [UX Network](https://uxnetwork.io?ref=genki-docs) in one place. If you want to learn more, our resources are full of guides, articles, news and up-to-date information. **Genki** is the place to go to learn all about Antelope based chains, [EOS](https://eosnetwork.com?ref=genki-docs), [Telos](https://www.telos.net?ref=genki-docs), [Wax](https://www.wax.io?ref=genki-docs) and [UX Network](https://uxnetwork.io?ref=genki-docs) in one place.

**[Genki](https://blog.genki.site/)** uses [Ghost](https://ghost.org?ref=docs-genki) open source technology, the easiest way to get a production instance deployed. It takes about 2 minutes to launch a new site with worldwide CDN, backups, security and maintenance.

[Ghost](https://ghost.org/?ref=docs-genki) is a powerful app for new-media creators to publish, share, and grow a business around their content. It comes with modern tools to build a website, publish content, send newsletters & more.
